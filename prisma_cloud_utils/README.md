## Installation

1. Clone the repository:
   ```bash
   git clone <TODO>
   cd coding-patterns
   ```

2. Create the virtual environment
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install dependencies
    ```bash
    pip install -r requirements.txt
    ```

### Requirements

- Python 3.7 or higher
- requests
- urllib3
